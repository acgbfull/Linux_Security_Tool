# Linux部分常见漏洞修复工具

## Description

本工具具有以下功能：
1. 隐藏php版本
2. 隐藏appache版本
3. 不显示Samba版本
4. 远端WWW服务支持TRACE请求漏洞修复
5. SSH 服务支持弱加密算法漏洞修复
6. SSL 3.0 POODLE攻击信息泄露漏洞
7. SSL/TLS 受诫礼(BAR-MITZVAH)攻击漏洞修复


## Operating environment

> CentOS（已测试）
> 
> Bash

>git clone https://github.com/acgbfull/Linux_Security_Tool.git
